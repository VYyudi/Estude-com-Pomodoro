# Estude com Pomodoro

Sistema web para estudos utilizando o método pomodoro 

## 🚀 Tecnologias

- **Frontend:** React, JavaScript

## 📦 Funcionalidades

- ✅ Timer Pomodoro
- ✅ Cadastro de matérias
- ✅ Histórico de sessões
- ✅ Meta diária
- ✅ Estatísticas

## 🎯 Como executar
```bash
npm install
npm start
```
